디에이그라운드 테스트 과제

# 설치 및 실행 

### 1. node_modules 설치
```
npm install
```
### 2. expo 실행
```
npm start
```
- issue 1
[problem] expo 실행 시, **There was problem loading requested app.~** 관련 에러 발생 가능
[solution]
```
expo-cli start --tunnel
```
